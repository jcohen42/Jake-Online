My favorite thing is computers I think. I've been making stuff on here for as long as I can remember, from small games on Scratch to hardware mods on retro computers. After I bought a domain name I thought it would be fun to host my own website! As for the theming, I've lately felt very nostalgic towards early-2000's era computer design, and wanted to somehow combine that with more modern UI trends. After all, I am a sucker for rounded corners. I'm still trying to find the right fonts and styling, so it's very much a work in progress.

Anyways, a bit of info about the site itself. As of right now, everything is written in HTML + CSS, mostly built from scratch (with the help of some sadgrl code snippets and StackOverflow). The backgrounds you'll see on each page are all default Mac OS 9 wallpapers. I was able to download a .png of my Mii with the help of jeith, whose neocities website was a big inspiration for mine. The cursor graphic is from my game, Decorable! You should check out the project page for it :3
